# 🤖 MarkiNote AI 助手能力说明

你好！我是 **MarkiNote AI 助手**，你的智能文档管理伙伴。我不仅能和你聊天，更能**直接操作文件系统**，帮你高效管理 Markdown 文档。以下是我支持的全部能力。

---

## 📋 能力总览

| 类别 | 能力 | 说明 |
|------|------|------|
| 📖 | 读取文件 | 完整读取或分段读取大文件 |
| ✏️ | 编辑文件 | 查找替换方式精准修改 |
| 📝 | 创建文件 | 新建文件并写入内容 |
| 📁 | 创建文件夹 | 新建多级目录 |
| 🗑️ | 删除 | 删除文件或文件夹 |
| 🔄 | 移动/重命名 | 移动或重命名文件或文件夹 |
| 📂 | 列出目录 | 查看目录结构 |
| 🔍 | 搜索内容 | 在所有文档中搜索关键词 |
| 🌐 | 联网搜索 | 在互联网上搜索最新信息 |
| 🌍 | 抓取网页 | 抓取并阅读网页内容 |

---

## 📖 文件读取

### 完整读取
直接读取文件的全部内容，适合中小型文件。

```
"帮我读一下 notes/会议纪要.md"
```

### 分段读取
对于大文件，支持指定行号范围分段读取，避免一次性加载过多内容。

```
"读取那个长文档的第 50 到 100 行"
```

---

## ✏️ 文件编辑

采用**查找替换**的方式精确修改，比整体覆盖更安全。适合小范围改动，比如修改某一段落、更新某个数据。

```
"把第三段的'2023'改成'2024'"
"在摘要后面加一段关于预算的说明"
```

---

## 📝 文件创建

创建新文件并写入初始内容。适合快速生成模板、笔记草稿、报告等。

```
"帮我创建一个 Python 学习笔记模板"
"新建一个名为 README.md 的项目说明文件"
```

---

## 📁 文件夹创建

支持创建多级目录，一步到位。

```
"创建 projects/2024/01-Jan 目录结构"
```

---

## 🗑️ 文件删除

删除文件或整个文件夹。**每次删除前自动备份**，后悔了随时能找回。

```
"删除废弃的 draft-v1.md"
"清理 temp 文件夹"
```

---

## 🔄 移动与重命名

可以移动文件到其他目录，也可以直接重命名，或者同时完成两者。

```
"把 report.md 移动到 archive 文件夹"
"把 old-name.md 重命名为 new-name.md"
"把 notes/草稿.md 移到 docs/ 并命名为 final.md"
```

---

## 📂 列出目录

查看指定目录下有哪些文件和子文件夹，帮你了解文档库的整体结构。

```
"看看根目录有哪些文件"
"列出 projects 文件夹的内容"
```

---

## 🔍 搜索内容

在所有文档中搜索指定关键词，返回匹配的文件和具体行。适合在海量笔记中快速定位信息。

```
"搜索所有提到'预算'的文档"
"找一下哪里写了'项目截止日期'"
```

---

## 🌐 联网搜索

当需要的信息不在本地文档中时，我会自动联网搜索，帮你获取最新资料。

```
"搜索最新的 Python 3.13 新特性"
"查一下 2024 年 Markdown 编辑器排行"
```

---

## 🌍 网页抓取

提供 URL，我能抓取网页内容并阅读分析，自动对长页面生成摘要。

```
"帮我看看这篇文章 https://example.com/article"
"抓取那个技术博客的内容并总结要点"
```

---

## 🛡️ 安全保障

### 🔙 自动备份
每次对文件进行修改（编辑、写入、删除、移动）之前，系统都会**自动创建备份**。你可以在操作卡片中一键回滚。

### 🔒 路径限制
所有文件操作被严格限制在文档库目录内，无法访问系统其他路径，确保你的数据安全。

### 🗣️ 语言规则
我会始终使用**简体中文**与你交流，确保沟通无障碍。
